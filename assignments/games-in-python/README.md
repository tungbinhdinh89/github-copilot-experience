
# 📘 Assignment: Hangman Game Challenge

## 🎯 Objective

Build a classic word-guessing game to practice string manipulation, loops, conditionals, and random selection in Python. Players will guess letters to reveal a hidden word before running out of attempts.

## 📝 Tasks

### 🛠️ Implement Hangman Game

#### Description

Create a Hangman game where players can guess letters one at a time to reveal a hidden word before exhausting their attempts. The game should provide clear feedback on progress and handle win/loss conditions.

#### Requirements

Completed program should:

- Randomly select words from a predefined list
- Accept letter guesses and show current progress (_ _ _ format)
- Track incorrect guesses remaining
- End when word is guessed or attempts exhausted
- Display win/lose messages
